Embed — Google Form (responsivo)

Cole o código do Google Forms dentro do <iframe>. Este layout torna o iframe responsivo e com altura proporcional para formulários longos.

Snippet para copiar

<div class="iframe-wrap">
  <iframe src="https://docs.google.com/forms/d/e/SEU_FORM_ID/viewform?embedded=true" title="Formulário — DIÁRIO DAS ATIVIDADES">Carregando…</iframe>
</div>

Observações rápidas:

  • Use o parâmetro ?embedded=true no src para a versão embutida.
  • Se o formulário ficar muito longo, aumente o padding-bottom em .iframe-wrap ou aplique uma altura fixa no próprio iframe.
  • Se quiser que o formulário abra em modal ou em nova aba, eu crio o script.
